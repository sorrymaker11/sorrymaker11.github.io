    # 图片

    插入图片有四种方式，分别是：

    • 插入本地图片
    • 插入网络图片
    • 文件使用HTML的img标签

    引入图片的基本格式是![描述](路径)，你可以选择适合自己的方式。

    # 插入网络图片

    语法如下（描述文字可不写）：

    ![描述图片的文字](https://cdn.pixabay.com/photo/2018/08/15/13/10/galaxy-3608029_1280.jpg)

    # 插入本地图片

    语法与插入网络图片类似，只需要换掉路径为本地路径即可

    # 使用HTML <img> 标签

    Markdown 中支持HTML 标签 来显示图片，当然也可以使用HTML 属性，来调节图片大小等：

    <img src="https://cdn.pixabay.com/photo/2018/08/15/13/10/galaxy-3608029_1280.jpg">

    # 链接

    # 超级链接：

    使用中括号和小括号表示，文字描述写在中括号内，网址写在小括号内，例如：

    这是百度网址 (opens new window)

    [这是百度网址](https://www.baidu.com)

    # 直接显示连接

    如果想要直接显示连接地址，也可以使用<> 符号将连接地址括住：

    https://www.baidu.com (opens new window)

    <https://www.baidu.com>

    # 无序列表可以使用+或-或* 表示，使用任意一种均可：

    + 无序列表1

    # 待办列表

    可以表示任务的处理状态，-[]表示待办状态，-[X] 表示已办状态： - [ ] 待办任务1 - [ ] 待办任务2 - [x] 已办任务

    • [ ] 待办任务1
    • [ ] 待办任务2
    • [x] 已办任务

    # 嵌套列表

    需要嵌套列表时，在子列表之前添加四个空格即可 - 无序列表 - 子列表是无序列表 - 子列表是无序列表

    # 表格

    用竖线|来分隔不同的单元格，使用 横线- 来分隔表头和其他行。

    |表头1  | 表头2 |表头3|
|--|--|--|
| 第1行第1列 | 第1行第2列 |第1行第3列|
| 第2行第1列 | 第2行第2列 |第2行第3列|

    # 效果

    表头1 表头2 表头3
    第1行第1列 第1行第2列 第1行第3列
    第2行第1列 第2行第2列 第2行第3列
